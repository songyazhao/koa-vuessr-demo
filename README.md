# Koa2.x + Vue2.x + Webpack4.x

> 一个基于Koa2.x + Vue2.x + Webpack4.x 的Vue服务端渲染方案

## 起步
```bash
yarn # npm install
```

## 开发

```bash
yarn dev # npm run dev
```

## 编译#部署

Node.js 7.6+

```bash
yarn build # npm run build
yarn start # npm start
```
